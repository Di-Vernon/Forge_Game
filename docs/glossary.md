# 검 만들기 게임 — 용어 사전

이 프로젝트에서 아래 정의와 다른 의미로 사용하지 않는다. 모든 문서와 코드에서 이 정의를 따른다.

## 게임 설계 용어

| 용어 | 정의 | 혼동 주의 |
|------|------|----------|
| **Block** | 레벨 그룹. Block 1(+0~+7) ~ Block 6(+25). 게임의 난이도 구간 분류. | 코드 블록, UI 블록과 무관. |
| **Phase** | 플레이어 여정 마일스톤. Phase A(+12 최초 도달) ~ Phase D(+25 달성). 유저 경험 구간. | 코드의 ForgePhase(강화 UI 상태 머신: idle→forging→result)와 **완전히 별개**. |
| **강화** | +n에서 +n+1로 레벨을 올리는 시도. 성공/실패 두 결과. 실패 시 보호 or 파괴. | |
| **제작** | 특정 레벨의 검을 만드는 행위. **강화 성공**(강화로 제작) + **조합소 제작** 두 경로 모두 포함. 칭호 획득 조건의 "N회 제작"은 양쪽 합산. | |
| **파괴** | 강화 실패 시 검이 +0으로 리셋되는 것. 보호가 발동하면 파괴가 아님(레벨 유지). | "실패"와 다름. 실패 = 강화 불성공. 파괴 = 실패 후 +0 리셋. 보호 발동 시 "실패했지만 파괴되지 않음". |
| **보호** | 칭호 효과로 파괴를 사전에 방지하여 현재 레벨을 유지하는 것. 불굴(35%), 검성(60%). | 재료 보존과 별개. 보호 = 레벨 유지. 검성 보호 시 재료도 보존. 재련의 정점은 보호 없이 재료만 보존. |
| **건너뛰기** | 칭호 부가효과. 골드+특정 조각 1개를 지불하여 강화 없이 즉시 해당 레벨의 검을 획득. 현재 레벨이 +0일 때만 사용 가능 (사용 후 레벨이 올라가므로 자연히 라운드당 1회). | 조합소 직접 제작과 다름. 건너뛰기는 칭호 장착 필요. |
| **라운드** | 강화소(ForgeScreen) 진입부터 퇴장까지의 단위. 하나의 검으로 진행하는 연속 강화 세션. 라운드 시작 시 칭호 잠금. | |
| **재료 검** | +17 이상 강화 시 투입해야 하는 하위 레벨 검 (≤+16). 비재귀적 — 재료 검 자체는 재료를 요구하지 않음. | |
| **조각** | 파괴 시 드랍되는 8종 소재 아이템. 조합소에서 스크롤/검 제작에 사용. | "조각"은 항상 이 8종 소재를 가리킴. 검 자체를 "조각"이라 부르지 않음. |
| **복원 스크롤** | 파괴 후 DestroyScreen에서 사용. 소멸된 검을 **파괴 직전 레벨로 복원** (파괴가 없었던 것처럼). | "보호"와 다름. 보호 = 파괴 사전 방지(레벨 유지). 복원 = 파괴 사후 복구(동일 레벨로 복원). 결과는 같지만 발동 시점이 다름. |
| **Effort** | 경제 모델의 기축통화. 1 Effort = 강화 시도 1클릭. 모든 자원의 가치를 환산하는 내부 단위. 유저에게 노출되지 않음. | @docs/economy_model.md 참조 |

## 코드 전용 용어

| 용어 | 정의 | 혼동 주의 |
|------|------|----------|
| **ForgePhase** | 강화 화면의 UI 상태 머신 (idle→forging→waiting_reveal→result). | 게임 설계의 Phase A~D와 **완전히 별개**. |
| **ForgeOutcome** | 강화 시도 1회의 결과 객체. `result`, `destroyed`, `protectionTriggered`, `protectionSource`, `newLevel` 포함. | |
| **protectionTriggered** | ForgeOutcome 필드. 불굴/검성 칭호 보호 발동 여부. 발동 시 destroyed=false, newLevel=현재 레벨 유지. | |
| **protectionSource** | ForgeOutcome 필드. 보호를 발동시킨 칭호 ID (`indomitable_smith` 또는 `sword_saint`). 미발동 시 null. | |
| **FragmentDropResult** | `{ fragmentId: FragmentId; count: number }`. `rollFragmentDrop`의 반환 원소 타입. 배열로 반환됨. | |
