### +8 인챈티드 소드

**컨셉**: 일반 롱소드에 가속 마법 룬이 각인됨. 룬에서 푸른 기운이 꿈틀거림.

**형태**:
- **날**: 직선 양날 롱소드 (길이 175px, 폭 18px). 깨끗한 은색. 정돈된 형태.
  - **풀러 1개** — 날 중앙, 길이의 75%.
  - Shine rect: 날 좌측 가느다란 하이라이트.
- **특수 — 룬 문자**: 도신 중앙, 풀러 위에 세로로 작은 룬 문자열 — `<text>` 또는 작은 기호(`rect`+`line` 조합) 5~6개, 세로 배치.
  - 룬 색: 파란색 (#4488dd), opacity 0.6.
  - **룬 glow 애니메이션**: 룬 문자 주위에 반투명 파란 `<path>`가 뱀처럼 꿈틀거리는 효과.
    - 구현: CSS `@keyframes rune-flow` — 불투명도 0.1↔0.4 + translateY 미세 이동 (±3px), 3s 주기.
    - 또는 GSAP으로: 룬 주위 path의 opacity + y 위치를 반복 애니메이션.
    - path 형태: 룬 문자를 따라 날 위에서 S자로 흐르는 가느다란 곡선 2줄, stroke #60a0ff, strokeWidth 1.5, opacity 변동.
- **가드**: 직선 가로막대 (폭 60px, 높이 10px). 은색. 가드 끝 약간 아래로 구부러짐.
