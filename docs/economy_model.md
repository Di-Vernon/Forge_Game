# 검 만들기 게임 — 경제 모델 (Effort 기반 가치 체계)

> **이 문서는 게임 내 모든 자원의 상대적 가치를 정량화하는 내부 도구이다.**
> 유저에게 노출되지 않으며, 밸런싱 결정의 근거로 사용한다.
>
> 관련 문서: @docs/balancing_status.md, @docs/knowledge_base.md
> 시뮬레이션 데이터: `simulation/v11/sim_step1_results.json`, `simulation/v11/economy_data.json`

## 1. 기축통화: Effort

### 정의

```
1 Effort = 강화 시도 1회 (= 1 클릭)
```

- 시간 환산: 1,200 Effort = 1시간 (20클릭/분 × 60분)
- 모든 자원의 가치를 Effort로 환산한다
- 골드 또한 Effort의 파생물이다 (Phase별 Effort/G 비율이 다름)

### 왜 골드가 아닌가

골드는 인플레이션이 있다:
- Phase A: 1,000G ≈ 강화 5~7회 가능 (큰 돈)
- Phase C: 1,000G ≈ 강화 0.1회 (푼돈)

Effort는 Phase에 무관하게 일정한 가치를 가진다.
1 Effort = "플레이어의 시간 1단위"이므로 체감 가치가 안정적.

## 5. 적용 사례

### 사례 1: 블러디 쇼텔(+16) 조합소 수량 결정 ✅ 확정

```
강화 Effort (보호 없는 플레이어 기준): E_farm(+16, 0%) = 1,386.2 E
광물파편 1개 Effort: 511.5 E (v2 실측, 드랍 테이블 변경 반영)
적정 비용 범위 (50~80%): 693.1 ~ 1,109.0 E

필요 수량: 693.1 / 511.5 ~ 1,109.0 / 511.5 = 1.35 ~ 2.17
→ 2개가 적정. 그러나 조합소는 리스크 회피 프리미엄 부과 → 6개 채택

6개 비용: 511.5 × 6 = 3,069 E
→ E_farm(+16, 0%) 대비 221% — 보호 없는 플레이어에게도 비쌈
→ E_farm(+16, 35%) = 348 E 대비 882% — 불굴 해금 후엔 강화가 훨씬 효율적

의도된 설계: 조합소 제작은 "보호 칭호 미해금 상태의 비상 루트"
→ 칭호 해금 후에는 강화로 전환하는 것이 자연스러운 플레이 흐름

현실적 제약:
→ 광물파편 게임당 획득 27.2개, 소비(건너뛰기+조합소) 27.7개 (근사 균형)
→ 6개 = 게임 전체 획득량의 22% — 1~2회 제작은 감당 가능
```

**결론: 블러디 쇼텔(+16) 조합소 레시피 = 알 수 없는 광물 파편 ×6 확정**

주 사용 대상: 보호 칭호 미해금 플레이어의 비상 루트 (강화 대비 비싸지만 파괴 리스크 회피)
칭호 해금 후: 강화(불굴 348 E, 검성 122 E)가 훨씬 효율적이므로 자연스럽게 강화로 전환

### 사례 3: 판매가 적정성 검증

> Phase별 골드 순 흐름이 양수 → 플레이어가 골드 부족으로 막히지 않음.
> 전체 잉여 +661,304G (게임당) → 판매가 현 수준 유지 가능 판정.
> 단, Phase C/D 건너뛰기 비용(83,359G / 63,882G)이 높아 스크롤 구매 여력 존재.

### 사례 4: 스크롤 가격 적정성 검증

> 구매 Effort 기준: Phase A 3.0 E (×1), Phase C 0.5 E (×1) → 매우 저렴.
> 조합소 제작 Effort 기준: 최저 265.8 E (마력이 부여된 철조각 ×3) → 구매가 53~530배 저렴.
> 결론: 현 스크롤 가격(150G/637G/1,125G) 적정. 조합소 제작은 "조각 과잉 처리용"으로만 의미.
